## 🚀 Getting Started

### Prerequisites

To run this project, you will need:

- [Rust](https://rust-lang.org/) installed on your machine.
- [Docker](https://www.docker.com/) installed on your machine.
- [Node](https://nodejs.org/en) installed on your machine.
- [Pnpm](https://pnpm.io/) installed on your machine.
- [Mise](https://mise.jdx.dev/) installed on your machine if you are using Mac / Linux.

> ⚠️ If you are a windows user, you don't need to install mise, you can run this project from source command.

### Run the project

Before your run this project, please take a look at [Configuration](#-configuration) part.

## ⚙️ Configuration

### Config frontend .env file from the root of the project

```shell
cp frontend/.env.example frontend/.env
```

### Config backend .env file from the root of the project

```shell
cp backend/.env.example backend/.env
```
